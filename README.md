*A simple MVVM Notes App using Kotlin*

## Main Features
- Create, Update, Delete, Search notes
- Add image & url to note

## Technologies Used:
- Kotlin
- View model + Live Data + Room Database
- Coroutine
- Navigation Component
- Navigation Safe Args
- Staggered RecyclerView
